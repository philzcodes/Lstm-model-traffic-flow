About Traffic Flow Prediction System

Traffic Flow Prediction System is a project designed to forecast traffic volume using machine learning.

The system utilizes an LSTM model trained on historical traffic data to predict future traffic flow.

This project aims to provide insights for traffic management and urban planning.

Technologies Used

  • HTML, CSS, JavaScript
  • Tailwind CSS
  • FastAPI
  • TensorFlow/Keras
